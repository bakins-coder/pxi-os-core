# PXI UNIFIED OS

A high-performance Unified Operating System designed for Catering, Finance, and AI-driven Service Management.

## Repository Fix
To resolve the "Failed to load file differences" error, run the automated repair script for your operating system:

### Windows (Command Prompt)
```cmd
SYNC_REPO.bat
```

### Mac / Linux / Git Bash
```bash
bash SYNC_REPO.sh
```

## Features
- **Unified CRM**: Pipeline management and customer tracking.
- **Catering Manager**: Specialized banquet forms, recipe costing, and automated logistics.
- **Finance Hub**: Real-time bookkeeping, requisitions, and automated collections.
- **AI Integration**: Gemini-powered operational agents and customer portal assistants.
- **Collaborative Suite**: Team chat, video calls, and AI meeting scribes.

## Tech Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Build Tool**: Vite 6
- **AI**: Google Gemini API (@google/genai)
- **Icons**: Lucide React
- **Charts**: Recharts

## Deployment
This project is optimized for deployment as a Progressive Web App (PWA) with built-in offline support.
