# Directive: Personal Assistant Dashboard Design

**To**: Nova (Web Visualization Specialist)
**From**: Prof (Orchestrator)
**Priority**: MEDIUM

We are moving into Phase 2 of the Personal Assistant project.

**Objective**:
Design a premium, high-fidelity web dashboard that will serve as the primary interface for Akin's PIM system.

**Design Goals**:
1.  **Premium Aesthetic**: Glassmorphism, subtle gradients, and reactive micro-animations.
2.  **Information Density**: Cleanly display Journal entries, CRM follow-ups, and Project progress.
3.  **Responsive**: Must look stunning on mobile (since Akin uses terminal-to-web access) and desktop.
4.  **Integration**: Anticipate data streams from Atlas's core database.

Please deliver initial wireframes and a design system overview to **Akin's Inbox** as `nova_result_pim_visual_concept.md`.
